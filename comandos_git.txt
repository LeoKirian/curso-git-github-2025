git init. 	-> cria um repo git na pasta atual
git status 	-> mostra o status do trabalho
git add nome_arquivo -> adiciona arquivo para ser commitado
git commit -m "msg" 	-> faz o commit do que estava para ser commitado
git log 	-> mostra todo o historio de commit
git diff nome_arquivo -> mostra o que foi alterado no arquivo desde o ultimo commit
git reset nome_arquivo -> remove após ter dado add

## FLUXO DE TRABALHO

1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. editar ou criar arquivos
4. git status
5. git add nome arquivo (ou git add . para a pasta toda)
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
